Flex Conteiner Align Items

O align-items é usado para posicionar os elementos verticalmente, seu efeito é melhor notado quando os elementos estão em pelo menos duas linhas. Os elementos devem estar dentro de uma div.pai e é essa div.pai que receberá o estilo de align-items.

flex-start

Flex-start é o valor padrão do align-items, os elementos ficam no começo do eixo-transversal.

flex-end

Já o flex-end se comporta de forma inversa ao flex-start. O flex-end posiciona os elementos no fim do eixo-transversal.

stretch

O stretch posiciona os elementos no topo do eixo-transversal, mas diferente do flex-start, o stretch também estica os elemento até preencher toda a altura do container. ATENÇÃO: para que o valor do stretch funcione, as div.filhas não podem ter um tamanho de height definido
